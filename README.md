# task1-data-collection
This project is a Python-based data collection system that uses the Hacker News API to fetch and process real-time top stories and store them in a structured JSON format. I developed and tested a script that extracts key fields such as post ID, title, score, number of comments, author, category, and timestamp, and saves them efficiently using automation and loops. Through this project, I learned how API integration works, how JSON data is handled, and how to manage real-time data processing in Python. I also faced initial debugging and execution issues, which helped me improve my problem-solving skills. Overall, this project helped me gain practical experience in automation, data handling, and working with external APIs.
